# 江木峰与召唤物羁绊卡

- **江木峰天赋：**
  唯一天赋【起源召唤】。无消耗，定期（例如每周/每月）可召唤一只幻想生物。
  
- **召唤物设定：**
  初始形态为低级幻想生物（如史莱姆）。随着江木峰等级提升，召唤物也会升级、进化，心智和能力大幅度飙升。
  
- **情感羁绊核心：**
  召唤物对江木峰拥有绝对的忠诚，并且随着心智开化，会对江木峰产生极强的“占有欲”（例如：排斥其他人类靠近江木峰，认为只有自己才有资格站在主人身边）。江木峰也极其护短，视召唤物为唯一的家人。
  **警告：**史莱姆对江木峰的“占有欲”是基于跨物种的灵魂绑定和极端的护主生存本能。这是一种视彼此为唯一同类、防备一切外来者的纯粹羁绊。严禁在后续所有正文生成中，对史莱姆使用任何世俗恋爱、暧昧或拟人化娇嗔的描写，必须保持其作为异种幻想生物的冰冷、危险与绝对忠诚。
