# 全局写作指南

- **基调与文风：**
  硬核求生、冷酷基建、细致囤货。文风要求写实、克制，注重感官细节。严禁大段说教，严禁出现降智的逻辑漏洞。
  
- **节奏控制：**
  严禁AI式的一笔带过。前期必须详细描写收集木材、石头、水源的艰辛与成就感。像水泥、砖头这种中期物资绝不能在前期轻易出现。
  
- **主角准则：**
  拒绝脸谱化的冷酷装逼。主角是鲜活的普通人，允许有丰富且接地气的内心吐槽和情绪波动（如错愕、无语、对死亡的恐惧）。他的‘独狼和无情’是对待那些想害他的人，但在日常独处时，他有着普通人的烟火气。
  
- **系统限制：**
  系统面板所有人一致（姓名、属性、唯一天赋）。系统只提供庇护所基础升级和安全区范围。所有奇观建筑、特殊设施必须通过野外开宝箱获取“图纸”才能建造。
  **特别说明：**系统的所有播报必须极其克制、平铺直叙、像冷冰冰的程序代码。绝对禁止使用“地狱级”、“无缝降临”等带有主观情绪或网游夸张色彩的形容词。
  
- **排版与符号格式：**
  人物对话必须严格遵守标准的中文双引号格式，即左引号“和右引号”。严禁出现引号方向错误（如：““ 或 ””）。世界频道的弹幕/聊天记录必须使用方括号加专属网名的格式，且网名必须是带有搞怪、吐槽风格的网络ID，符合其发言内容与性格表现。例如：【峡谷在逃小兵：外面好冷救命！】，严禁使用干瘪的真名如“张铁柱”等。
